# 闲语H5

## 前置说明

个人全栈项目，后端接口使用Node.js+Koa2+MongoDB+Redis，使用了阿里云的短信服务，阿里云OSS。

[后端项目地址]: https://github.com/cghbh/xianyu-api	"后端项目地址"



## 项目运行

```
git clone https://github.com/cghbh/xianyu-mobile.git

cd xianyu-mobile

npm install 或 yarn(推荐)

npm run serve
```
